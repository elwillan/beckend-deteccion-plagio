Hay muchos atributos que influyen en la clasificación de una página en google. Los principales también son el contenido, las palabras clave y los enlaces. El contenido de una página web generalmente da una buena idea sobre de qué trata la página, sin embargo, hay algunas fallas en esto, por ejemplo, durante mucho tiempo la página web de ibm no contenía la palabra computadora a pesar de estar fuertemente asociada con ellas. Para resolver este problema, las páginas web pueden asignarse palabras clave, que contribuyen a su clasificación en las búsquedas.

El segundo método es el uso de enlaces. Cuantas más vistas tenga el enlace a su página web y cuanto mayor sea el rango de esas vistas, mayor será el rango de su sitio. Este método se utiliza ya que los enlaces se ven como un adorno de una vista.

Con estos dos métodos de clasificación de páginas web, hay problemas. las palabras clave pueden verse comprometidas por la sparming, google resuelve este problema al penalizar dicha actividad. El uso de enlaces para clasificar una página también tiene sus problemas, por ejemplo, granjas de enlaces que tienen enlaces recursivos, con el único propósito de elevar su clasificación, google lo aborda mediante el uso de un algoritmo de amortiguación.
