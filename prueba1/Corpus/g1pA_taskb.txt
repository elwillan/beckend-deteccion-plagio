El algoritmo de PageRank se utiliza para designar cada aspecto de un conjunto de documentos hipervinculados con una ponderación numérica. El motor de búsqueda de Google lo utiliza para estimar la importancia relativa de una página web de acuerdo con esta ponderación. El sistema utiliza la distribución de probabilidad para determinar las probabilidades de que una persona que haga clic al azar en los enlaces llegue a cualquier página. Después de esto, cada página web recibe una clasificación de 0-10 según su relevancia para una búsqueda. El PageRank se calcula teniendo en cuenta el número de enlaces entrantes y el PageRank de las páginas que proporcionan estos enlaces. Por lo tanto, esto significa que si una página web está vinculada a otras que tienen una clasificación alta, entonces también recibirá una clasificación alta.

Debido a la naturaleza del sistema PageRank, es susceptible de manipulación y ha sido explotado para que ciertas páginas reciban una clasificación falsa y exagerada. En estos casos, solo Goggle tiene acceso al PageRank genuino. Sin embargo, se han realizado muchas investigaciones sobre métodos para evitar enlaces de documentos con un PageRank falso para tratar de solucionar los errores en este sistema y desde 2007 Google ha penalizado activamente los esquemas que intentan aumentar artificialmente las clasificaciones.
